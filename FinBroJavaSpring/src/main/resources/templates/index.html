User API Documentation

Attribute Data Type
Name String
Surname String
Age Integer
Username String
Email String
Password String

POST /api/users/addUser

Adds a new user.

Request Body: JSON representation of User object.

Responses:

  • 201 Created: User successfully added.
  • 400 Bad Request: If the request body is invalid or user already exists.

GET /api/users/allUsers

Retrieves all users.

Responses: List of User objects.

GET /api/users/getByUserId/{userId}

Retrieves a user by their ID.

Path Variable: userId (integer).

Responses:

  • 200 OK: User found and returned.
  • 404 Not Found: If user with the specified ID does not exist.

GET /api/users/getByUsername/{username}

Retrieves a user by their username.

Path Variable: username (string).

Responses:

  • 200 OK: User found and returned.
  • 404 Not Found: If user with the specified username does not exist.

PUT /api/users/updateUser

Updates an existing user.

Request Body: JSON representation of User object (Including UserID).

Responses:

  • 201 Created: User successfully updated.
  • 400 Bad Request: If the request body is invalid or user already exists.

DELETE /api/users/deleteByUserId/{userId}

Deletes a user by their ID.

Path Variable: userId (integer).

Responses:

  • 204 No Content: User successfully deleted.
  • 404 Not Found: If user with the specified ID does not exist.
