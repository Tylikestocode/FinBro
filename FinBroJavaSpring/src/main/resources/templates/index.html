User API

http://testvps.us.to/api/users

Path: /addUser

HTTP Method: PUT

Description: Creates a new User object and stores it in the database

Request Body Example:

{
    "name": "Emily",
    "surname": "Brown",
    "age": 24,
    "username": "emily.b",
    "email": "emily.b@example.com",
    "password": "hashed_password3"
}

Response Example:

{
    "id": 3,
    "name": "Emily",
    "surname": "Brown",
    "age": 24,
    "username": "emily.b",
    "email": "emily.b@example.com",
    "password": "hashed_password3"
}

* id field not needed in request. If set, will be ignored

Path: /validateCredentials

HTTP Method: POST

Description: Checks if a user exists with given email and password. Returns the User if matching credentials are found

Request Body Example:

{
    "email": "emily.b@example.com",
    "password": "hashed_password3"
}

Response Example

{
    "id": 3,
    "name": "Emily",
    "surname": "Brown",
    "age": 24,
    "username": "emily.b",
    "email": "emily.b@example.com",
    "password": "hashed_password3"
}

Path: /updateUser

HTTP Method: PUT

Description: Updates an existing User object

Request Body Example:

{
    "id": 3,
    "name": "New_Name",
    "surname": "New_Surname",
    "age": 30,
    "username": "new_Username",
    "email": "new_email@example.com",
    "password": "New_Password"
}

Response Example:

{
    "id": 3,
    "name": "New_Name",
    "surname": "New_Surname",
    "age": 30,
    "username": "new_Username",
    "email": "emily.b@example.com",
    "password": "New_Password"
}

Path: /deleteById/{userId}

HTTP Method: DELETE

Description: Deletes User object with given id

Request Path Example:

http://testvps.us.to/api/users/deleteById/3

Response Example:

{
    "message": "User successfully deleted"
}

Accounts API

http://testvps.us.to/api/accounts/

Path: /addAccount

HTTP Method: PUT

Description: Creates a new Account object and stores it in the database

Request Body Example:

{
    "name": "Alice Savings",
    "balance": 1000.00,
    "dateCreated": "2024-01-01 10:00:00",
    "minimumBalance": 500.00,
    "notes": "Primary savings account",
    "userId": 1,
    "categoryId": 1
}

Response Example:

{
    "id": 1,
    "name": "Alice Savings",

    "dateCreated": "2024-01-01 10:00:00",

    "categoryId": 1
